# What you need to have
- Install npm
    - https://www.npmjs.com/package/npm
- Install expo cli
    - https://docs.expo.dev/workflow/expo-cli/

# What you need to set
- helpers/http.js
    - serverURL (from VM instances - External IP)

# run
- npm install
- build
    - Android
        - `expo build:android`
    - IOS
        - `expo build:ios`
